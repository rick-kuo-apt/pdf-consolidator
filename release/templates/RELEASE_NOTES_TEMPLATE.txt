GETTING STARTED
---------------
1. Extract the ZIP to any local folder (not a network path)
2. Double-click PDFConsolidator.exe
3. Drag PDF files onto the window
4. Click "Merge PDFs"

STORAGE LOCATIONS
-----------------
Settings: %APPDATA%\PDFConsolidator\settings.json
Logs:     %APPDATA%\PDFConsolidator\app.log

SUPPORT
-------
If you encounter issues:
1. In the app, click Help > About / Support
2. Click "Export Support Bundle"
3. Share the generated ZIP file with your IT team

The support bundle contains sanitized logs and settings only.
No PDF files or passwords are ever included.
