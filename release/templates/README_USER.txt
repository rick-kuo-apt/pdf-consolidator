PDF Consolidator v{VERSION}
===========================

Merge multiple PDF files into a single document.

GETTING STARTED
---------------
1. Extract this ZIP to any folder (Desktop, Documents, etc.)
2. Double-click PDFConsolidator.exe
3. Drag PDF files onto the window
4. Click "Merge PDFs"
5. Done! Your merged PDF is saved.

TIPS
----
- Drag files to reorder them before merging
- Use Ctrl+O to add files, Ctrl+M to merge
- Settings are saved automatically
