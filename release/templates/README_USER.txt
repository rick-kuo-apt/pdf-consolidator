PDF Consolidator v{VERSION}
===========================

Merge multiple PDF files into a single document.

GETTING STARTED
---------------
1. Extract this ZIP to a local folder (Desktop, Documents, Downloads)
2. Double-click PDFConsolidator.exe
3. Drag PDF files onto the window
4. Click "Merge PDFs"
5. Done! Your merged PDF is saved.

TIPS
----
- Drag files to reorder them before merging
- Use Ctrl+O to add files, Ctrl+M to merge
- Settings are saved automatically

TROUBLESHOOTING (If It Won't Open)
----------------------------------
- IMPORTANT: Extract and run from a LOCAL folder (Desktop/Documents).
  Do NOT run directly from SharePoint, OneDrive, or a network path.

- If Windows blocks the app: Click "More info" then "Run anyway".
  Or: Right-click the exe > Properties > check "Unblock" > Apply.

- Still not working? In the app: Help > About / Support > Export Support Bundle.
  Send the ZIP file to your IT team or the maintainer.
